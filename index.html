01

Slash Commands

/ask <question>
Send any natural language query to the LLM and get a formatted response posted back to the channel.
/ask Explain Docker containers
/alert <message>
Post a structured alert notification to the current channel with sender context and timestamp.
/alert Production API is down
/summarize <text>
Summarize any block of text into 2–3 concise bullet points using LLM inference.
/summarize <paste meeting notes>

@BotMention — Mention the bot directly in any channel to ask a question without using a slash command. Responds inline with the same formatted Block Kit output.

Key Code

app.py — /ask command handler

app.py
@app.command("/ask")
def handle_ask(ack, respond, command):
  ack()  # acknowledge immediately
  user_query = command.get("text", "").strip()
  respond(f":hourglass: Thinking about: *{user_query}*...")
  answer = ask_llm(user_query)
  respond(blocks=[
    {"type": "section", "text": {"type": "mrkdwn",
     "text": f":mag: *Query:* {user_query}"}},
    {"type": "divider"},
    {"type": "section", "text": {"type": "mrkdwn",
     "text": f":robot_face: *Answer:*\n{answer}"}}
  ])

llm.py — swappable LLM backend

llm.py
def ask_llm(prompt: str) -> str:
  api_key = os.environ.get("OPENAI_API_KEY", "")
  base_url = os.environ.get("LLM_BASE_URL", "https://api.openai.com/v1")
  model = os.environ.get("LLM_MODEL", "gpt-3.5-turbo")

  payload = {
    "model": model,
    "messages": [{"role": "system", "content": "You are a helpful Slack assistant."},
                {"role": "user", "content": prompt}],
    "max_tokens": 500, "temperature": 0.7
  }
  return requests.post(f"{base_url}/chat/completions", ...) # → parsed response

Setup and Usage

terminal
# Clone and install
git clone https://github.com/sadhanageddam27/slack-ai-assistant-bot.git
pip install -r requirements.txt

# Configure credentials
cp .env.example .env
# Add SLACK_BOT_TOKEN, SLACK_APP_TOKEN, OPENAI_API_KEY

# Run
python app.py

Local LLM with Ollama — Run fully offline by setting LLM_BASE_URL=http://localhost:11434/v1 and LLM_MODEL=llama3 in your .env file. No OpenAI API key needed.
